# Visual Studio Code

Visual Studio Code is a lightweight but powerful source code editor. While we recommend using Visual Studio for C#/C++ development, VS Code is excellent for Lua development in FiveM.

## Download

You can download Visual Studio Code from their [official website](https://code.visualstudio.com/).

## Essential Extensions

### Lua Development

- [Lua](https://marketplace.visualstudio.com/items?itemName=sumneko.lua) - Provides core Lua language support
- [CfxLua (FiveM/RedM) IntelliSense](https://marketplace.visualstudio.com/items?itemName=overextended.cfxlua-vscode) - Adds FiveM-specific features:
  - Auto-completion
  - Type declarations
  - Syntax highlighting
  - Code annotations

### Recommended Extensions

To enhance your development experience, consider these additional extensions:

#### Visual Customization

- [Bearded Icons](https://marketplace.visualstudio.com/items?itemName=BeardedBear.beardedicons) - Beautiful file icons
- [Nord Deep Theme](https://marketplace.visualstudio.com/items?itemName=marlosirapuan.nord-deep) - A clean, modern theme

#### Development Tools

- [GitLens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens) - Enhanced Git integration
  - Requires [Git](https://git-scm.com/downloads) to be installed
